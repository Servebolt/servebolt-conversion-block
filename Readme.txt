This is plugin will let you insert dynamic conversion blocks into the content of a post to encourage clicks.

This project is maintained on [Github](https://servebo.lt/scb).

== Installation ==
1. Download this plugin and unzip
1. Upload servebolt-conversion-blocks folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

With WP-CLI
1. Log in to your site with SSH
2. cd to your sites root wordpress folder
3. Run `wp plugin install servebolt-conversion-blocks --activate`
